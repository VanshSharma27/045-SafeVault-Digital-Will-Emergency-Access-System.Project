# 🔐 SafeVault – Digital Will & Emergency Access System

SafeVault is a full-stack secure web application that allows users to safely store passwords, private notes, and important documents with authentication and secure access control.

This project demonstrates authentication, authorization, secure storage, and backend integration using Node.js and MongoDB.

---

## 🚀 Features

### 🔑 Authentication System
- User Registration
- Secure Login using JWT
- Password Hashing using bcrypt
- Protected Dashboard (Login Required)
- Logout Functionality

### 🔐 Secure Vault
- Add Secret (Title + Password/Note)
- View Saved Secrets
- Search Vault Items
- Show/Hide Secret
- Copy Password to Clipboard

### 🔑 Password Generator
- 12-character strong password generator
- Auto-fill into secret field

### 📁 Document Locker
- Upload files (PDF, Images, etc.)
- Preview uploaded files
- Stored securely in server

### 🚨 Emergency Contact System
- Save emergency contact email
- Concept of inactivity-based access (extendable)

### 📊 Activity Log
- Tracks user actions
- Logs secret save, password generation, file upload, etc.

### 🌙 UI Features
- Dark / Light Mode Toggle
- Modern Dashboard UI
