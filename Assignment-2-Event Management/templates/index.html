What is EventHub?

EventHub is a fully functional Event Management Portal built with HTML5, CSS3, JavaScript and Python Flask. It lets administrators create, edit and delete events while attendees can browse, search, filter and register — all backed by an SQLite database.

  • ✅ Dynamic Jinja2 templates
  • ✅ Responsive CSS Grid layout
  • ✅ Client-side JS validation
  • ✅ Flask CRUD + session-based admin
  • ✅ Live search & category filter
  • ✅ RSVP counter per event
